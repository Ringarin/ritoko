works

うさとたび

担当部分
全て（LP・バナーデザイン / 運用更新ディレクション /コーディング）
制作ツール
HTML5 / CSS3 / JavaScript / jQuery / Illustrator / Photoshop
制作期間
10日間
対応デバイス
PC / スマートフォン
作品概要
大分県宇佐市で農村民泊ツアーを提供している観光協会の新規Webサイト
制作目的
農村民泊の認知度UP、ツアーの利用客増加
→地域貢献（人口減少地域の活性化・農業従事者の増加に繋げる）
ターゲット
農業に少しでも関心がある家族・夫婦
コンセプト
「うわぁ、行きたい！」と思ってもらえるような、わくわく感を大切にしたサイト。
あくまで体験ツアーとして、敷居を低くできるような、親しみやすいサイト。
サイト名
ロゴ
宇佐市の「うさ」と、故郷の「さと」、旅の「たび」を合わせてサイト名に。
円の内側と外側で色が反転していることから、農泊体験によって、農業、農村、農作物に対しての見方が大きく変わるということを表現。
