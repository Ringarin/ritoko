制作ツール
HTML5 / CSS3 / JavaScript / jQuery / Illustrator / Photoshop
制作期間
9日間（3名で制作）
対応デバイス
PC / タブレット / スマートフォン
作品概要
不動産販売会社の新築マンションPRサイト
※他グループがクライアントとして希望した要件を元に、実際にミーティングを実施。作成したデザインカンプを元にすり合わせ後、サイト作成。
制作目的
新築マンションのPR・認知度の向上、モデルルームの集客へ繋げる
ターゲット
30代のファミリー層（共働きで世帯年収1000万円）
コンセプト
高級感漂う、落ち着いた雰囲気のサイト。ワンランク上のライフステージを提供できることを表現し、資料請求率の向上につなげる。
サイト名
ロゴ
サイト名はクライアントチームより指定。ロゴは、4つの図形を重ねることで地上40階建てを表現。
